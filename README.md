# Airbnb-mocksite!
<ul><b>Airbnb-mocksite</b> is a UI mocksite of the famous website Airbnb.</ul>


# Features


<ul>
 <li>Data changes dynamiclly on each reload</li>
 <li>Working sliders to filter home page data</li>
 <li>Fully responsive design</li>
</ul>
